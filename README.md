# Phortivia Website

A responsive one-page website for **phortivia.com**, positioned as a modern freight brokerage focused on proactive communication, vetted capacity, and dependable shipment execution.

## Files

- `index.html` — page structure and copy
- `styles.css` — responsive design and animation
- `script.js` — mobile navigation, scroll effects, and demo form behavior
- `favicon.svg` — browser icon / compact brand mark

## Publish

Upload the folder to any static host, including Netlify, Vercel, Cloudflare Pages, GitHub Pages, or standard cPanel hosting.

## Replace before launch

1. Replace `+1 (000) 000-0000` in `index.html`.
2. Confirm `hello@phortivia.com` and `carriers@phortivia.com` exist.
3. Connect the quote form to a form handler, email service, or CRM.
4. Add applicable legal business details and operating-authority information.
5. Add Privacy Policy and Terms pages if collecting personal information.
